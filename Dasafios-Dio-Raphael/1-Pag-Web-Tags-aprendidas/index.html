Algumas Tags que aprendi

  1. Tag "a"

    Esta TAG define um hiperlink, que é usado para vincular de uma página a outra. O atributo mais importante do elemento (< a >) é o atributo href, que indica o destino do link. Por padrão, os links aparecerão da seguinte forma em todos os navegadores: Um link não visitado está sublinhado e em azul. Um link visitado está está sublinhado e em roxo. Um link ativo está está sublinhado e em vermelho

  2. Tag "font"

    A tag < font > foi usada no HTML 4 para especificar o tipo da fonte, o tamanho da fonte e a cor do texto.

  3. Minha cor favorita é azul vermelha!
